# ML-DLprojects

## Chest Xray Classification
- Multi-class classification where each X-ray image is assigned to one pathology class from set of possible thoracic pathologies.

## Comment Category Prediction
-  analyze a dataset containing information about short textual entries submitted to an online discussion system. The dataset includes the content of each entry, engagement signals from other users, and outputs from automated analysis components.
-  build predictive models that can accurately determine the final category assigned to each comment

## Plant leaves super resolution
- building a Conditional Generative Adversarial Network (cGAN) to perform a blind 4x Super-Resolution upscale. By taking the degraded 32 X 32 inputs and reconstruct the high-frequency biological textures (veins, chlorosis, and necrotic lesions) to output pristine 128 X 128 images.
